Este documento descreve como lidar com vulnerabilidades e relatórios de segurança para o **LegendsHub Manager**.

As seguintes versões do software recebem suporte e patches de segurança ativos:

| Versão | Suportada | Notas |
| :--- | :---: | :--- |
| **1.0.x** |  | Versão atual estável (React & Electron) |

## Como Reportar uma Vulnerabilidade

Se você descobrir alguma vulnerabilidade de segurança neste jogo (seja no motor de simulação local ou no empacotamento eletrônico), **por favor, não crie uma issue pública**. Siga as etapas abaixo para fazer um reporte responsável:

1. Envie um e-mail descrevendo detalhadamente a vulnerabilidade para o e-mail oficial do mantenedor: **Davisonsant@gmail.com**
2. Se possível, anexe provas de conceito (PoC), prints de tela ou passos para reproduzir o problema.
3. Você receberá uma resposta de confirmação em até **48 horas**.

Agradecemos profundamente por nos ajudar a manter o LegendsHub seguro para toda a comunidade de jogadores!
